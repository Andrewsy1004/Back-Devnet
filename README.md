# DevNet Backend 

1. Clone the Project 
2. ```npm install```
3.  Clone the file ```.env.template``` file and rename it to ```.env```
4. Update the environment variables as needed
5. Start the database using Docker
```
docker-compose up -d
```

6. Start ```npm run start:dev```
